      Quick Start
    </h2>
    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get up and running with the Action List Opportunity component in minutes
    </p>
  </div>

  <Steps>
    <Step title="Deploy to Salesforce">
      Clone the repository and deploy the Lightning Web Component to your Salesforce org using SFDX CLI.

      ```bash
      git clone https://github.com/dev98reynaga/Action-List-Opportunity.git
      cd Action-List-Opportunity
      sfdx force:source:deploy -p force-app/main/default
      ```
    </Step>

    <Step title="Add Component to Page">
      Add the `operationDatatable` component to your Lightning page using the Lightning App Builder. The component will automatically fetch Opportunity records.

      ```xml lightning-page-metadata.xml
      <componentInstance>
        <componentName>c:operationDatatable</componentName>
      </componentInstance>
      ```
    </Step>

    <Step title="Configure Filtering">
      Select a filter field from the dropdown (Name or StageName) and use the search input to filter records. The component supports both server-side and client-side filtering.

      <Note>
        The search functionality is disabled until you select a filter field from the dropdown menu.
      </Note>
    </Step>

    <Step title="Manage Selections">
      Select rows using checkboxes, then click the "Selected" button to view only your selected records. Click "Records" to return to the full dataset.

      <Tip>
        Selected rows are preserved when switching between filtered and full views, making it easy to build and manage your working set.
      </Tip>
    </Step>
  </Steps>
</div>
